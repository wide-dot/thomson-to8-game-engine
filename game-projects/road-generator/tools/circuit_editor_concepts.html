🏁 Lotus Circuit Editor — Concepts d'édition multi-segments + générateur procédural

document de conception · non fonctionnel · pour choisir ce qu'on implémente

A · Édition multi-segments — modifier plusieurs sections simultanément

A1 · Proportional Editing

★ recommandé

Inspiré de Blender (O / Proportional Edit). Chaque modification du segment central se propage aux voisins avec un poids décroissant défini par une courbe de falloff.

Paramètres : rayon (nb de segments affectés de chaque côté), force, type de falloff : Smooth (cosinus), Sharp (x²), Linear, Bell (gaussienne), Constant.

✓ Intuitif, connu des utilisateurs Blender ✓ 3 paramètres seulement, prévisible ✓ Visualisation claire dans le plan et la 3D ✗ Symétrique par défaut (falloff identique gauche/droite) → idéal pour sculpter courbes + pentes progressivement

A2 · Sélection de plage + tampon de forme

★ recommandé

Inspiré du vectoriel (Illustrator, Inkscape). On sélectionne une plage de segments [début..fin] via drag sur le scrubber ou shift+clic dans le plan, puis on applique une forme prédéfinie (chicane, bosse, épingle, S-curve) avec amplitude et symétrie.

La forme est une fonction paramétrée qui remplace (ou additionne à) les valeurs dans la plage.

✓ Crée des formes reconnaissables en 2 clics ✓ Bibliothèque extensible (pré-sets Lotus réels) ✗ Moins organique — résultat "géométrique" → idéal pour créer des sections typiques (chicane, épingle)

A3 · Pinceau de sculpture (Sculpt Mode)

complexité moyenne

Inspiré de ZBrush / Mudbox (sculpt) et des outils calligraphie vectorielle. Un pinceau positionné sur le circuit applique l'effet au survol/glissement. Modes : Courbure+/-, Pente+/-, Lisser (moyenne voisins), Bruit.

Taille = nb de segments, force = amplitude, falloff = courbe d'atténuation radiale.

✓ Très intuitif, sensation de sculpture directe ✓ Résultat organique, non répétitif ✗ Résultats difficiles à reproduire exactement ✗ Le "lissage" peut être nécessaire après sculpture brute → idéal pour l'exploration créative et le raffinement

A4 · Édition par keyframes éparses (spline)

★ recommandé

Inspiré des splines Bézier / Catmull-Rom (Blender F-Curves, After Effects). On ne place que quelques keyframes clés (10–15 pour 100 segments), la courbe entre eux est interpolée automatiquement. Poignées tangentes Bézier pour la forme locale.

C'est conceptuellement l'inverse de l'édition actuelle : au lieu de modifier chaque segment, on sculpte la courbe en entier via des points de contrôle clairsemés.

✓ Très puissant pour de grandes sections ✓ Courbes naturellement lisses, continuité C1/C2 ✗ Changement de paradigme (plus de segment direct) → idéal pour concevoir le circuit de zéro, complémentaire du A1

A5 · Enveloppe de modulation

utilitaire

Inspiré de l'automation audio (Pro Tools, Ableton) et des filtres vectoriels. Une courbe d'enveloppe (peu de points) est posée au-dessus du canal courbure ou pente. Le résultat final = signal original × enveloppe (ou + offset).

Utile pour : atténuer une section progressivement, amplifier un virage, appliquer une symétrie, ou créer un fade-in/out d'effet.

✓ Non-destructif (original conservé) ✓ Puissant pour la mise à l'échelle globale ✗ Indirection — moins direct que A1 ou A3 → outil de post-traitement, complémentaire

B · Générateur procédural — partir d'une base plutôt que d'une ligne droite

B1 · Profils de circuits + paramètres

★ recommandé

6 profils de base : Ovale, Technique, Montagne (élévation), Street (angles droits), Flowing (courbes douces), Hillclimb (montée linéaire). Chaque profil = une recette paramétrique (courbes + pentes) adaptée aux contraintes Lotus.

Paramètres : intensité des virages, relief, symétrie, graine aléatoire, nb de segments. Génère directement dans le circuit courant (ou nouveau).

✓ Départ concret, pas une ligne droite ennuyeuse ✓ Profils calibrés pour les contraintes int8 ✓ Reproductible (graine) + variabilité → combiné avec A1+A3 pour affiner après génération

B2 · Grammaire de tronçons (assemblage)

intermédiaire

Inspiré des grammaires de forme et des éditeurs de niveaux à briques (Mario Maker, Track Mania). Le circuit est une séquence de sections nommées (droite, arc G/D, épingle, chicane, bosse...) avec des paramètres par section.

Un solveur de fermeture mesure en temps réel combien de courbure il reste pour boucler (Σδ_curve × 16 substeps = −2048) et ajuste automatiquement.

✓ Compréhensible (liste de sections nommées) ✓ Solveur de fermeture garantit la boucle ✗ Moins flexible pour les formes hybrides → idéal pour des circuits structurés, lisibles

B3 · Générateur aléatoire contrôlé (graine)

rapide à implémenter

Génération par graine (seed) reproductible : la même graine donne toujours le même circuit. Chaque profil = un ensemble de règles (distribution des virages, amplitude, fréquence des bosses) + une dose de bruit déterministe (Perlin ou LCG).

Boutons : ↺ Rejouer (nouvelle graine aléatoire), → Adopter (charge dans l'éditeur pour affinage). Rapide à parcourir, conserve les bons résultats.

✓ Exploration rapide, reproductible ✓ Simple à implémenter (LCG + règles) ✓ Découverte : circuit inattendu mais correct ✗ Nécessite un affinage systématique après → idéal pour prototyper vite 10 idées de circuit

C · Recommandation — combinaison optimale pour votre contexte

Recommandation : implémenter A1 + B1/B3 en priorité, puis A3.

A1 (Proportional Editing) — le plus polyvalent et le mieux connu des artistes. Un slider de rayon + un dropdown de falloff suffisent. Visualisation dans le plan (dégradé de couleur sur les segments affectés). S'intègre directement dans le panneau Segment actuel. Implémentation : 2–3 jours.

B1 + B3 (Profils + graine) — résout le problème du départ en ligne droite. 6 profils paramétrables + génération reproductible. Le générateur utilise notre pipeline Python (source de vérité). Implémentation : 2–3 jours.

A4 (Splines) — mode d'édition alternatif très puissant pour la forme globale, mais change le paradigme (inverse de l'édition segment-par-segment). À ajouter après A1/B1 comme couche optionnelle.

A3 (Pinceau) — complément naturel de A1 pour le lissage et la sculpture organique. Facile à ajouter une fois A1 en place (mêmes primitives). Implémentation : 1–2 jours après A1.

✓ A1 + B1 résolvent 80% du besoin en une semaine ✓ Chaque outil reste cohérent avec le modèle int8 Lotus → A5 (enveloppe) et B2 (grammaire) : plus tard si besoin
